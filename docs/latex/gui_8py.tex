\doxysubsection{Detailed Description}
Graphical User Interface (GUI) for SLCP Chat. 

This module provides a complete graphical frontend for the SLCP (Simple LAN Chat Protocol) client. It allows users to send and receive messages and images, manage peer visibility, toggle AFK (away-\/from-\/keyboard) mode, access configuration settings, and quit the session gracefully.

Communication with the network and discovery processes is handled via multiprocessing pipes. Built using Py\+Qt5 and styled optionally using QDark\+Style.

Key GUI Features\+:
\begin{DoxyItemize}
\item Display chat log with timestamps and color-\/coded messages
\item Input fields for recipient and message
\item Send image button (opens file dialog)
\item View active clients
\item AFK mode toggle with autoreply functionality
\item Dark mode toggle
\item In-\/app configuration management
\end{DoxyItemize}
