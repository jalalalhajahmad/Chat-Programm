A decentralized, peer-\/to-\/peer chat application developed in Python for the {\bfseries{Betriebssysteme und Rechnernetze}} (Operating Systems and Computer Networks) course. ~\newline
 This project implements a custom protocol called SLCP (Simple Local Chat Protocol) to demonstrate real-\/time communication over local networks without relying on central servers.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Project Overview}{Project Overview}}\label{index_autotoc_md1}
The application enables direct communication between clients in the same network, supporting both text messaging and image transfer. Peers are discovered dynamically via UDP broadcasting. The system combines lightweight protocols (UDP for control/data, TCP for binary files) and provides both a graphical and terminal-\/based user interface.

Typical use cases\+:
\begin{DoxyItemize}
\item Real-\/time 1\+:1 peer messaging
\item Reliable image file transfer
\item Away-\/from-\/keyboard (AFK) status with autoreplies
\item Auto-\/discovery of connected clients in the network
\item GUI and CLI interface options
\end{DoxyItemize}

\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{Peer Discovery\+:}} Broadcast-\/based peer discovery using {\ttfamily JOIN}, {\ttfamily WHO}, and {\ttfamily KNOWUSERS} messages.
\item {\bfseries{Message Exchange\+:}} Real-\/time message delivery over UDP.
\item {\bfseries{Image Transfer\+:}} TCP-\/based file transfer with UDP notification handshakes.
\item {\bfseries{AFK Mode\+:}} Automatic autoreplies when a user is away.
\item {\bfseries{Graphical Interface\+:}} Built using Py\+Qt5 with dark/light theme support.
\item {\bfseries{Settings Dialog\+:}} Runtime configuration for user handle, port, autoreply message, and image folder.
\item {\bfseries{CLI Interface\+:}} Text-\/based command-\/line chat interface with full feature parity.
\item {\bfseries{Fully Documented\+:}} Comprehensive technical documentation generated using Doxygen.
\end{DoxyItemize}
