\doxysubsection{Detailed Description}
Peer discovery module for SLCP (Simple LAN Chat Protocol) 

This module implements the decentralized peer discovery mechanism used by the SLCP chat application. It uses UDP broadcasting to detect other clients on the same network via {\ttfamily JOIN}, {\ttfamily WHO}, and {\ttfamily KNOWUSERS} messages. One process takes the role of a WHO responder and provides a list of all known clients upon request.

The discovery process is run in a separate process and periodically sends discovery messages to update the local peer list.
