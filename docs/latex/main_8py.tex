\doxysubsection{Detailed Description}
Main graphical launcher for the SLCP (Simple Local Chat Protocol) peer-\/to-\/peer chat application. 

This script serves as the entry point for launching the GUI version of the SLCP client. It reads configuration from a TOML file, initializes inter-\/process communication pipes, and starts three key processes\+:
\begin{DoxyItemize}
\item {\ttfamily discovery\+\_\+process} (UDP broadcast peer discovery)
\item {\ttfamily network\+\_\+process} (handles SLCP messages, AFK state, and file/image transfers)
\item {\ttfamily gui\+\_\+process} (Py\+Qt5 graphical chat interface)
\end{DoxyItemize}

Main responsibilities\+:
\begin{DoxyItemize}
\item Validates user input and config structure
\item Manages subprocess startup and shutdown
\item Coordinates inter-\/process communication
\item Persists updated configuration settings to {\ttfamily config.\+toml}
\end{DoxyItemize}
